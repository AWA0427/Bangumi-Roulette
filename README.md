# Bangumi-Roulette

一个基于 Bangumi API 的番剧抽签转盘应用
（还在调试中，目前对UI界面不太满意，尝试重构）



## 项目文件结构

├── main.py                  # 主程序入口

├── requirements.txt # 依赖包列表

├── README.md # 项目说明

└── ui/

   ├── init.py

   ├── components.py # UI组件（按钮、卡片、头像等）

   ├── history_tab.py # 历史记录标签页

   ├── settings_tab.py # 设置标签页（用户认证、筛选设置）

   ├── theme.py # 主题管理系统

   └── wheel_tab.py # 轮盘抽签标签页
